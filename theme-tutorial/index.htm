Installing

You can install the theme in 2 ways: By using the ZIP file downloaded from the WordPress Theme Directory and uploading it via your WordPress Admin or using the WordPress Theme installer.

This procedure will install the Storyboard Comic theme on your WordPress site. You can activate the theme after it has been uploaded.

WordPress Theme Installer

This method will download and install the theme directly from the WordPress Theme Directory

  1. Login into your WordPress Admin section (http://www.domain.com/wp-admin)
  2. Go to Appearance > Theme
  3. Click on the Add New button, next to the "Manage Themes" heading
  4. Search for "Storyboard Comic"
  5. Click "install" and then click "Install Now"

Using the ZIP file

The theme will come as a packaged ZIP file. You can use this archive to install the theme on your WordPress site. To install your theme, do the following:

  1. Login into your WordPress Admin section (http://www.domain.com/wp-admin)
  2. Go to Appearance > Theme
  3. Click on the Add New button, next to the "Manage Themes" heading
  4. Click on Upload
  5. Click on Browse and look for the downloaded ZIP file
  6. Click on Install Now

Advanced Method: FTP Upload

The ZIP file can also be extracted and uploaded the /wp-content/themes/ via FTP. This method will vary depend on your hosting provider.

Activate Theme

Once the theme is uploaded and installed, you can activate the theme, goto Appearance > Theme and click "Activate" on the theme.

Adding Galleries

Galleries can be one of two types:

List
Very similar to WordPress standard pages except will show child galleries under the content.
Player
Will show a fullscreen gallery with uploaded images.

There can be any combinations of parent/child galleries but player pages will not show child pages.

Frontpage Panels

The theme uses WordPress menu editor, when activated the theme will automaticaly add a menu called 'Frontpage Panels', so all you need to do is add items to the menu.
